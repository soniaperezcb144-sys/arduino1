02 — CONCEPTO

¿Cómo Funciona?

Un potenciómetro es una resistencia variable de 3 terminales. Al girar su eje, cambia la tensión que entrega al pin analógico del Arduino (entre 0V y 5V). Arduino convierte ese voltaje en un valor digital de 0 a 1023 con analogRead(). Luego, con la función map() se escala ese rango a 0–255, y finalmente analogWrite() genera una señal PWM en el pin del LED para controlar su brillo.

Señal PWM

  • Pulsos de encendido/apagado a alta frecuencia (~490 Hz)
  • Duty cycle del 0% = LED apagado
  • Duty cycle del 100% = LED al máximo
  • Pines PWM en Arduino: 3, 5, 6, 9, 10, 11

Potenciómetro 10kΩ

  • Terminal izquierdo → 5V (VCC)
  • Terminal derecho → GND
  • Terminal central (wiper) → Pin A0
  • Divide el voltaje proporcionalmente al giro

Función map()

  • Convierte un rango numérico en otro
  • Entrada: 0–1023 (analogRead)
  • Salida: 0–255 (analogWrite / PWM)
  • Sintaxis: map(valor, 0, 1023, 0, 255)

03 — CIRCUITOS

Diagramas e Implementaciones

IMG 01

Circuito con Batería 9V y Potenciómetro

En esta implementación se utiliza una batería de 9V como fuente de alimentación independiente. El potenciómetro azul de 10kΩ actúa como divisor de voltaje: su terminal central entrega un voltaje variable que controla la corriente que fluye a través de la resistencia de protección y el LED rojo. Al girar el potenciómetro, la tensión cambia y con ella el brillo del LED. La resistencia en serie (color marrón/negro) protege al LED de sobrecorriente. Este esquema es ideal para entender el control analógico antes de integrar Arduino.

IMG 02

LEDs en Paralelo con Arduino UNO

Tres LEDs rojos conectados en paralelo a diferentes pines digitales del Arduino UNO. Cada LED tiene su propia resistencia limitadora de corriente. Esta configuración permite controlar cada LED de forma independiente, lo que es la base para aplicar PWM individual a cada uno. Los cables negros retornan al GND común del Arduino.

05 — PROGRAMACIÓN

Código Arduino

El sketch lee el valor analógico del potenciómetro en el pin A0, lo mapea al rango PWM (0–255) y escribe ese valor en el pin 9 para controlar el brillo del LED.

circuito_leds.ino 
// C++ code
//
void setup()
{
  pinMode(13, OUTPUT);   // LED 1
  pinMode(12, OUTPUT);   // LED 2
  pinMode(11, OUTPUT);   // LED 3
}

void loop()
{
  // Enciende los 3 LEDs
  digitalWrite(13, HIGH);
  digitalWrite(12, HIGH);
  digitalWrite(11, HIGH);
  delay(1000);             // Espera 1 segundo

  // Apaga los 3 LEDs
  digitalWrite(13, LOW);
  digitalWrite(12, LOW);
  digitalWrite(11, LOW);
  delay(1000);             // Espera 1 segundo
}

Pines Utilizados

  • Pin 13 — salida digital LED 1
  • Pin 12 — salida digital LED 2
  • Pin 11 — salida digital LED 3
  • GND — tierra común del circuito

Funciones Clave

  • pinMode(pin, OUTPUT) — configura el pin como salida
  • digitalWrite(pin, HIGH) — enciende el LED (5V)
  • digitalWrite(pin, LOW) — apaga el LED (0V)
  • delay(1000) — pausa de 1 segundo entre estados
